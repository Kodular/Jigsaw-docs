## Installation
<Steps>
1. Install NodeJS from https://nodejs.org. Any latest or LTS version will work.

2. Open the terminal, and run the following command to install pnpm:
    ```sh title="Install pnpm"
    npm i -g pnpm
    ```

3. Install the Jigsaw app using the installer.
    <CardGrid>
        <LinkCard title="Download Jigsaw" href="/#download" />
    </CardGrid>
</Steps>

## Usage
Try out these steps:
<Steps>
1. After successful installation, open the Jigsaw app.

2. From the Home screen, click on the New button in the top-right corner to create a new project. It will ask you to specify a project name, and hit Save. This will create a new Folder under your home directory. 

    <Tabs>
    <TabItem label="Windows" icon="seti:windows">
        `C:\Users\{user name}\JigsawProjects\{project name}`
    </TabItem>
    <TabItem label="macOS / Linux">
        `/home/{user name}/JigsawProjects/{project name}`
    </TabItem>
    </Tabs>

    :::caution
    It is currently not possible to customize the path.
    :::

3. Then, click on the newly created project to open the project editor.

4. Drag the blocks as shown there to get started.

5. Then, click on the Preview button on the top-right corner to start an emulator to preview your app. This opens a new window titled "Jigsaw Preview".

6. Change the blocks to play around and see the preview change in real-time.
</Steps>
